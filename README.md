# Проект: МMesto

### Описание

Сервисом Mesto представляет собой интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки.
Проект является учебным материалом, предназначенным для закрепления навыков по Html, CSS и изучению новых материалов по JavaScript.
В проект можно добавлять свои фотографии, увеличивать фото для просмотра, удалять фотографии и лайкать их. 
Можно редактироавть аватар, добавлено подтверждение при удалении фотографии.
Реализована валидация форм.
Проект подключен к серверу. 

## Технологии
* Grid Layout
* Flexbox-вёрстка
* Использование JavaScript
* Webpack
* API
